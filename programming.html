Lập Trình Python: Từ Cơ Bản Đến Nâng Cao

Python - Ngôn Ngữ Của Tương Lai

Python không chỉ là một ngôn ngữ lập trình, mà còn là chìa khóa mở ra cánh cửa đến với công nghệ hiện đại. Từ phát triển web, phân tích dữ liệu, đến trí tuệ nhân tạo, Python đều tỏa sáng nhờ cú pháp dễ hiểu và hệ sinh thái thư viện phong phú.

Bài viết này sẽ dẫn dắt bạn từ những bước đầu tiên với Python đến các kỹ thuật nâng cao, kèm theo các dự án thực tế để bạn áp dụng ngay.

Bắt Đầu Với Python

Để sử dụng Python, bạn cần cài đặt nó:

# Kiểm tra cài đặt Python
python3 --version
# Hoặc
python --version

Sau khi cài đặt, bạn có thể chạy Python từ terminal hoặc sử dụng IDE như VS Code, PyCharm để viết code.

Chương Trình Python Đầu Tiên

Hãy viết một chương trình đơn giản để làm quen:

# Chương trình chào hỏi
name = input("Nhập tên của bạn: ")
print(f"Chào mừng {name} đến với TechBit!")
for i in range(3):
    print(f"Lần chào thứ {i+1}")

Biến và Kiểu Dữ Liệu

Dưới đây là cách khai báo và sử dụng các kiểu dữ liệu cơ bản:

# Các kiểu dữ liệu
integer = 42
float_num = 3.14
string = "Python"
list_data = ["Code", "Learn", "Build"]
dict_data = {"language": "Python", "version": 3.11}

print(f"Số nguyên: {integer}")
print(f"Số thực: {float_num}")
print(f"Chuỗi: {string}")
print(f"Danh sách: {list_data}")
print(f"Từ điển: {dict_data}")

Điều Khiển Luồng Chương Trình

Sử dụng if, for, và while để điều khiển logic:

# Điều kiện
temperature = 30
if temperature > 35:
    print("Trời quá nóng!")
elif temperature > 25:
    print("Trời ấm áp.")
else:
    print("Trời mát mẻ.")

# Vòng lặp for
for num in range(1, 6):
    print(f"Số: {num}")

# Vòng lặp while
countdown = 5
while countdown > 0:
    print(f"Đếm ngược: {countdown}")
    countdown -= 1

Tạo Hàm Thú Vị

Hàm giúp tái sử dụng code một cách hiệu quả:

# Hàm tính diện tích hình tròn
def circle_area(radius):
    pi = 3.14159
    return pi * radius ** 2

print(f"Diện tích hình tròn (r=5): {circle_area(5)}")
print(f"Diện tích hình tròn (r=10): {circle_area(10)}")

Dự Án: Trò Chơi Đoán Số

Hãy thử lập trình một trò chơi đoán số đơn giản:

# Trò chơi đoán số
import random

def guess_game():
    secret_number = random.randint(1, 100)
    attempts = 0
    print("Chào mừng đến với trò chơi đoán số (1-100)!")

    while True:
        guess = int(input("Nhập số bạn đoán: "))
        attempts += 1
        if guess < secret_number:
            print("Số lớn hơn!")
        elif guess > secret_number:
            print("Số nhỏ hơn!")
        else:
            print(f"Chúc mừng! Bạn đoán đúng sau {attempts} lần.")
            break

guess_game()

Thư Viện Python Đỉnh Cao

Dưới đây là một số thư viện bạn nên biết:

Thư viện Công dụng Cài đặt
NumPy Tính toán số học pip install numpy
Pandas Xử lý dữ liệu pip install pandas
Requests Gửi yêu cầu HTTP pip install requests

Ví dụ sử dụng Requests để lấy dữ liệu từ web:

# Lấy dữ liệu từ API
import requests

response = requests.get("https://api.github.com")
print(response.status_code)  # 200 = Thành công
print(response.json())  # Dữ liệu JSON
